# 요구사항 분석 (Requirements)

이 폴더는 프로젝트의 비즈니스 요구사항과 기능 명세를 담고 있습니다.

## 문서 목록

### PROJECT_REQUIREMENTS.md
프로젝트의 전체 요구사항을 분석한 문서입니다.

**주요 내용:**
- 프로젝트 개요 및 목적
- 비즈니스 컨텍스트
- 3가지 핵심 기능 상세 요구사항
  - 시간대별 예약 가격 세팅
  - 추가상품 관리
  - 예약 총 가격 정보 저장 및 계산
- 도메인 규칙 (Invariants)
- 비기능 요구사항
- 성공 지표

**대상 독자:**
- 프로젝트 매니저
- 개발자
- QA 엔지니어
- 비즈니스 분석가

**사용 시점:**
- 프로젝트 시작 전 요구사항 확인
- 새로운 기능 추가 시 기존 요구사항 참조
- 테스트 케이스 작성 시 검증 기준 확인
